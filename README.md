# BondStats Monetary System Map™

An interactive, static GitHub Pages project for explaining how the modern monetary system works.

## What it includes

- Interactive monetary system map
- Clickable nodes with explanations
- Highlighted connection flows
- Search and category filters
- Animated money-flow mode
- Light/dark mode
- Links to BondStats Learn pages
- No APIs
- No external libraries
- No third-party graphics

## How to publish on GitHub Pages

1. Create a new GitHub repository.
2. Upload all files from this folder.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /root
5. Save.
6. GitHub will publish the project as a public URL.

## How to embed or link from BondStats

Recommended use:

- Add a button on BondStats: `Open the Monetary System Map`
- Link to the GitHub Pages URL.
- Add internal links from related Learn pages.

## Customization

Edit `app.js` to change:

- Node explanations
- BondStats Learn page links
- Search keywords
- Connected guide cards

Edit `styles.css` to change:

- Colors
- Layout
- Node styling
- Mobile behavior
